org.omg.CORBA public final class FREE_MEM extends SystemException Exception thrown when the ORB failed in an attempt to free dynamic memory, for example because of heap corruption or memory segments being locked. It contains a minor code, which gives more detailed information about what caused the exception, and a completion status.
