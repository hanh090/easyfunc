java.lang.management public interface BufferPoolMXBean extends PlatformManagedObject The management interface for a buffer pool, for example a pool of direct or mapped buffers. A class implementing this interface is an MXBean. A Java virtual machine has one or more implementations of this interface. The getPlatformMXBeans method can be used to obtain the list of BufferPoolMXBean objects representing the management interfaces for pools of buffers as follows: List pools = ManagementFactory.getPlatformMXBeans(BufferPoolMXBean.class); The management interfaces are also registered with the platform MBeanServer. The ObjectName that uniquely identifies the management interface within the MBeanServer takes the form: java.nio:type=BufferPool,name=pool name where pool name is the name of the buffer pool.
