java.awt.event public abstract class ComponentAdapter extends Object implements ComponentListener An abstract adapter class for receiving component events. The methods in this class are empty. This class exists as convenience for creating listener objects. Extend this class to create a ComponentEvent listener and override the methods for the events of interest. (If you implement the ComponentListener interface, you have to define all of the methods in it. This abstract class defines null methods for them all, so you can only have to define methods for events you care about.) Create a listener object using your class and then register it with a component using the component's addComponentListener method.
