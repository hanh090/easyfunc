javax.naming.spi public class DirectoryManager extends NamingManager This class contains methods for supporting DirContext implementations. This class is an extension of NamingManager. It contains methods for use by service providers for accessing object factories and state factories, and for getting continuation contexts for supporting federation. DirectoryManager is safe for concurrent access by multiple threads. Except as otherwise noted, a Name, Attributes, or environment parameter passed to any method is owned by the caller.
