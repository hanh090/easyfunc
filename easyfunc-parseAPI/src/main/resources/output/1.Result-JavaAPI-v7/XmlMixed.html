javax.xml.bind.annotation @Retention(value=RUNTIME) @Target(value={FIELD,METHOD}) public @interface XmlMixed Annotate a JavaBean multi-valued property to support mixed content. The usage is subject to the following constraints: can be used with @XmlElementRef, @XmlElementRefs or @XmlAnyElement The following can be inserted into @XmlMixed annotated multi-valued property XML text information items are added as values of java.lang.String. Children element information items are added as instances of JAXBElement or instances with a class that is annotated with @XmlRootElement. Unknown content that is not be bound to a JAXB mapped class is inserted as Element.
