javax.xml.bind.annotation @Retention(value=RUNTIME) @Target(value={FIELD,METHOD,PARAMETER}) public @interface XmlAttachmentRef Marks a field/property that its XML form is a uri reference to mime content. The mime content is optimally stored out-of-line as an attachment. A field/property must always map to the DataHandler class.
