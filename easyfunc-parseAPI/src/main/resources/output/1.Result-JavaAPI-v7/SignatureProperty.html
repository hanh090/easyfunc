javax.xml.crypto.dsig public interface SignatureProperty extends XMLStructure A representation of the XML SignatureProperty element as defined in the W3C Recommendation for XML-Signature Syntax and Processing. The XML Schema Definition is defined as: A SignatureProperty instance may be created by invoking the newSignatureProperty method of the XMLSignatureFactory class; for example: XMLSignatureFactory factory = XMLSignatureFactory.getInstance("DOM"); SignatureProperty property = factory.newSignatureProperty (Collections.singletonList(content), "#Signature-1", "TimeStamp"); String getTarget() Returns the target URI of this SignatureProperty. String getId() Returns the Id of this SignatureProperty. List getContent() Returns an unmodifiable list of one or more XMLStructures that are contained in this SignatureProperty. These represent additional information items concerning the generation of the XMLSignature (i.e. date/time stamp or serial numbers of cryptographic hardware used in signature generation).
