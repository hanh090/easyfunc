javax.management.remote public class NotificationResult extends Object implements Serializable Result of a query for buffered notifications. Notifications in a notification buffer have positive, monotonically increasing sequence numbers. The result of a notification query contains the following elements: The sequence number of the earliest notification still in the buffer. The sequence number of the next notification available for querying. This will be the starting sequence number for the next notification query. An array of (Notification,listenerID) pairs corresponding to the returned notifications and the listeners they correspond to. It is possible for the nextSequenceNumber to be less than the earliestSequenceNumber.
